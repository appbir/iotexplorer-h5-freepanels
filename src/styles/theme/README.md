## 二、公共部分
公共部分主要分【公共变量】和【公共样式类】两类
### 2.1 公共变量
> 1. 变量的目的降低多主题间的代码重复量
> 2. 不满足变量的规则的范围在主题内进行重置或覆盖

1. 原子变量 
>  定义样式的最小变量

2. 便捷变量
>  快速开发变量

3. 产品变量
> 不同产品独有变量
### 2.1 公共样式类
> 1. 系统级公共样式类(跨主题)--base>common.less
> 2. 产品级公共样式类--主题文件夹下的>common.less
